# DiadramVision
Very often in the practice of working with paper documents, we are faced with the task of converting a graph into a computer format, but is it long and difficult to use expensive programs for digitization? Dia Vision is designed just for such cases. This is an application that scans a graph through a smartphone camera or using a graphical interface on a computer, finds guides and saves in a convenient format. Such an application is necessary for scientists working with big data, Data-Scientist specialists, businessmen, secretaries, office workers and others. Dia Vision allows you to save a lot of time and concentrate more on your work.
